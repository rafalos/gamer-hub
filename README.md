# Gamer HUB

Games library powered by RAWG API to browse, add to library, mark as favourite and track games.

## Tech stack

- Typescript
- NextJS 15 (app router)
- PostgreSQL + drizzle ORM
- BetterAuth
- Bull MQ (for background games fetching)
- Tailwind
- Shadcn components
- Redis
- Zustand
- Zod
- React Form Hook
